# FUTURE_CS_01
# Cyber Security Internship – Task 1

## Vulnerability Assessment Report

This project contains a vulnerability assessment conducted on the OWASP Juice Shop web application using a passive, read-only approach.

### Target
OWASP Juice Shop  
http://juice-shop.herokuapp.com

### Tools Used
- Nmap
- OWASP ZAP
- Browser Developer Tools
- Kali Linux

### Key Findings
- Content Security Policy (CSP) Header Not Set
- Cross-Domain Misconfiguration
- Vulnerable JavaScript Library
- Strict-Transport-Security Header Not Set
- Server Information Disclosure
- X-Content-Type-Options Header Missing

### Report
The full vulnerability assessment report is included in this repository.
